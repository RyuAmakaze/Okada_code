## フィルタの特徴
使用したローパスフィルタはベッセルフィルタと楕円フィルタです。
<br>

### ベッセルフィルタの特徴
- 位相をできるだけ崩しません。
- パワースペクトルが緩やかにしか減衰しません。
<br>

### 楕円フィルタの特徴
- 位相は崩れてしまいます。根拠は、群遅延がフラットにならないからです。
- パワースペクトルは急峻に減衰します。急峻に減速するため、cpdのパワースペクトルを重視した従来研究では好まれていたことが予想されます。
