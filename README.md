## Imageフォルダ

## PowerSpectrum.ipynb
Imagesフォルダにあるアルファベット画像のパワースペクトルと位相をクロスさせる．
MIX(A,B)と指定すれば，  
位相はAスペクトルはA，  
位相はAスペクトルはB，  
位相はBスペクトルはB，  
位相はBスペクトルはA，  
を表示する．
